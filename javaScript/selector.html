# 元素选择器

常用的选择器除标签、id、元素及class外，还有层级选择器，层级选择器可以在给定祖先元素下匹配所有后代，例如$(“form input”)；子类选择器如$(“form > input”)；获取div标签后面的input元素：$(“div +input”)；选择同辈：$(“from ~ input”)；获取第一个元素：$(“li:first”),与之对应的有last，选择最后一个；或者筛选选择器，选择所有未选中的元素$(“input : not(:checked)”)；根据索引值获取元素,获取索引值为偶数的元素$(“tr:even”)；与之对应的odd为查找奇数元素；也可以查找给定索引值元素,获取索引值为1的元素$(“tr:eq(1)”)；获取比给定索引大或小的元素$(“tr:gt(1)”)/$(“tr:lt(2)”)； 伪类选择器$(“:header”)获取所有标题，获取正在执行动画的元素$(“$div:not(:animated)”)；根据字符串查找元素$(“div:contains(‘A’)”);查找不包含子元素或文本的空元素$(“td:empty”)，$(“td:parent”)查找所有含子元素或文本的元素；匹配含有选择器所匹配元素的元$(“div:has(p)”)；查找所有隐藏的元素$(“tr:hidden”),visible为可见元素；还有一些特殊匹配值的选择器。
