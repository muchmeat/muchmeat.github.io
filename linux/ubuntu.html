# 修改ubuntu默认用户

  • 找到ubuntu.exe

C:\Users\用户名\AppData\Local\Microsoft\WindowsApps

ubuntu config --default-user root

# 添加 sudoers

问题

xxxx is not in the sudoers file.  This incident will be reported.

使用root编辑 vim /etc/sudoers

vim /etc/sudoers

找到 root ALL(ALL:ALL) ALL 在下面增加一行，只是用户名不同即可
