# springboot

# 定时任务

主要由三种实现方式

# 1.基于注解 @Scheduled

基于注解@Scheduled默认为单线程，开启多个任务时，任务的执行时机会受上一个任务执行时间的影响

# 如何启用多线程

  1. 首先，在 application.yml 或 application.properties 配置文件中添加以下内容
spring:
  task:
    scheduling:
      pool:
        size: 10 # 设置线程池大小
  1. 在定时任务的类中添加 @EnableScheduling 注解，并在需要执行定时任务的方法上添加 @Scheduled 注解

@Component
@EnableScheduling
public class MyTask {

    @Scheduled(fixedRate = 5000)
    public void doSomething() {
        // 执行定时任务的代码
    }

    /**
     * 定时任务，每5分钟，同步消息
     */
    @Scheduled(cron = "0 0/5 * * * ?")
    public void doSomething2() {
        // 执行定时任务的代码
    }
}

Cron表达式参数分别表示：

秒（0~59） 例如0/5表示每5秒
分（0~59）
时（0~23）
日（0~31）的某天，需计算
月（0~11）
周几（ 可填1-7 或 SUN/MON/TUE/WED/THU/FRI/SAT）
