# docker部署vuepress

参考 https://www.cnblogs.com/Aamir-Ye/p/16831571.html

# vue项目打包dist文件夹上传到Ubuntu服务器

首先在Ubuntu服务器上创建文件夹，作为项目根目录。将windows系统内dist文件拷贝到刚刚创建的文件夹。

/mnt挂载目录，可以在Linux内访问到windows系统内文件

cp /mnt/d/study/vuepress-blog/docs/.vuepress/dist .

# 创建Dockerfile

# 1.在项目根目录创建名为Dockerfile的文件

touch Dockerfile

# 2.编辑Dockerfile文件

sudo vim Dockerfile

内容如下

# 指定基础镜像
FROM nginx:latest

# 将本地的dist文件夹复制到容器中的指定位置
COPY dist/ /usr/share/nginx/html

# 容器暴露的端口号
EXPOSE 80

# 容器启动时运行的命令
CMD ["nginx", "-g", "daemon off;"]

# 构建Docker镜像

在项目根目录下执行

sudo docker build -t your-image-name .

your-image-name 替换为你想要的镜像名称

# 运行Docker容器

构建完成后，运行以下命令来启动Docker容器

sudo docker run -d -p 8080:80 your-image-name

其中 -d 代表后台运行。建议服务启动有问题时移除 -d 再执行
