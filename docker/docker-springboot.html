# docker部署springboot

参考 (opens new window)

# springboot项目打包jar文件上传到Ubuntu服务器

首先在Ubuntu服务器上创建文件夹，作为项目根目录。将windows系统内jar文件拷贝到刚刚创建的文件夹。

sudo mkdir springbootVue

/mnt挂载目录，可以在Linux内访问到windows系统内文件

sudo cp /mnt/d/study/code/springbootVue/target/springbootVue-1.0-SNAPSHOT.jar .

# 创建Dockerfile

# 可以在Ubuntu服务器上创建编辑

# 将windows上创建好的文件拷贝到Ubuntu

sudo cp /mnt/d/study/vuepress-blog/docs/docker/Dockerfile .

内容如下

# FROM: 基础镜像，基于jdk8镜像开始
FROM openjdk:8

# COPY: 将应用的配置文件也拷贝到镜像中。
COPY *.jar /app.jar

CMD ["--server.port=8082"]

 # EXPOSE：声明端口
EXPOSE 8082

# ENTRYPOINT：docker启动时，运行的命令，这里容器启动时直接运行jar服务。
ENTRYPOINT ["java","-jar","/app.jar"]

# 构建Docker镜像

在项目根目录下执行

sudo docker build -t springbootvue .

# 运行Docker容器

构建完成后，运行以下命令来启动Docker容器

--sudo docker run -d -p 8082:8082 springbootvue
sudo docker run -p 8082:8082 springbootvue

其中 -d 代表后台运行。建议服务启动有问题时移除 -d 再执行

# 测试是否能访问

sudo curl localhost:8082

# windows浏览器测试是否能访问

一般是访问localhost即可。但有时会出现无法访问的现象。可通过以下方式访问

访问地址的IP可以通过命令查看一下

sudo ifconfig

通过该方式访问可能会导致docker build 镜像异常，机器重启就好了~~~
