# docker 镜像发布到阿里云

参考 (opens new window)

登录阿里云账号

# 创建命名空间

在 容器镜像服务 内找到命名空间模块，创建命名空间

可以将仓库设置为公共

# 创建镜像仓库及上传镜像

按照需要，选择相应的 命名空间，填写仓库信息，然后下一步中代码源选择本地仓库，然后点击 创建镜像仓库

然后就能看到仓库的基本信息了,参考里面的操作指南进行操作

主要是以下操作

docker images

docker login --username=账号 registry.cn-hangzhou.aliyuncs.com

docker tag [ImageId] registry.cn-hangzhou.aliyuncs.com/命名空间/blog:[镜像版本号]

docker push registry.cn-hangzhou.aliyuncs.com/命名空间/blog:[镜像版本号]

# 镜像版本管理

上传镜像完成后可以查看镜像版本

# 拉取镜像

sudo docker pull registry.cn-hangzhou.aliyuncs.com/命名空间/blog:[镜像版本号]

# 启动镜像

sudo docker run -d -p 8080:80 registry.cn-hangzhou.aliyuncs.com/命名空间/blog:[镜像版本号]
