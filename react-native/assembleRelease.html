# 打包过程出现问题

# 1.node_modules 下 blacklist.js 出现 Unterminated character class

Invalid regular expression: /(.\fixtures\.|node_modules[\]react[\]dist[\].|website\node_modules\. |heapCapture\bundle.j s|.\tests\.)$/: Unterminated character class

解决办法 将blacklist.js内

var sharedBlacklist = [
    /node_modules[/\\]react[/\\]dist[/\\].*/,

    /website\/node_modules\/.*/,

    /heapCapture\/bundle\.js/,

    /.*\/__tests__\/.*/
];

其中的

/node_modules[/\\]react[/\\]dist[/\\].*/

修改为

/node_modules[\/\\]react[\/\\]dist[\/\\].*/
