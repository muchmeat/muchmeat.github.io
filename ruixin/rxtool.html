# 基础平台工具使用总结

# 实体内关联关系的配置

关联关系建立在当前实体与关系实体间，当前实体可以作为一方，关系实体可以作为一方，这样两个实体间就会产生关系。当实体间是一对一关系时，需要配置是否依存对方，当是一对多关系时需要配置对方是否多个，当是多对多关系时会内部产生中间表

# 事件内逻辑配置之动作流

动作流是一系列逻辑的组织，包括前端交互，服务器端数据处理。需要关注的逻辑的组织
